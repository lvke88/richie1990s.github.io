Java基本数据类型和包装类(1)–包装类父类

Java是面向对象的语言，但Java中的基本数据类型却是不面向对象的。为了解决不足，在设计类的时候为每个基本数据类型设计了一个对应的类进行代表，这样和基本数据类型对应的类称为 包装类(Wrapper Class) 。基本数据类型和包装类对应的关系如下：

基本数据类型 包装类
byte Byte
boolean Boolean
short Short
char Character
int Integer
long Long
float Float
double Double

Number–包装类的父类

抽象类 Number 是 AtomicInteger, AtomicLong, BigDecimal, BigInteger, Byte, Double, DoubleAccumulator, DoubleAdder, Float, Integer, Long, LongAccumulator, LongAdder和 Short 类的超类。Number 的子类必须提供将表示的数值转换为 byte、double、float、int、long 和 short 的方法。
