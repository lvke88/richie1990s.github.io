Sublime Text 3使用(2)-安装常用插件

下面对常用插件的安装进行说明，常用插件有 Package Control、 IMESupport、 OmniMarkupPreviewer 等

Package Control安装

打开Sublime Text3，按 Ctrl+` 或者 View > Show Console 打开命令窗口，粘贴以下代码并回车即可。

import urllib.request,os; pf = ‘Package Control.sublime-package’; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), ‘wb’).write(urllib.request.urlopen( ‘http://sublime.wbond.net/‘ + pf.replace(‘ ‘,’%20’)).read())

上述代码只适用于Sublime Text3

IMESupport安装

安装IMESupport插件必须先安装Package Control插件。按 Ctrl+Shift+p 或者 Preferences > Package Control 弹出窗口，选择 Package Control: Install Package ，再次在新弹出的窗口输入IMESupport，选择 IMESupport 即可自动安装。安装完成后中文输入法就会跟随光标了。

OmniMarkupPreviewer安装

OmniMarkupPreviewer插件是支持Markdown标记语言的，可以实时预览，同时支持导出Html源码文件。安装过程和上文的IMESupport安装是一致的。按 Ctrl+Shift+p 或者 Preferences > Package Control 弹出窗口，选择 Package Control: Install Package ，再次在新弹出的窗口输入OmniMarkupPreviewer，选择 OmniMarkupPreviewer 即可自动安装。
Ctrl+Alt+o 可以在浏览器中预览Markdown文件。
